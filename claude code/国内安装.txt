windows操作系统安装claude code

1.安装git

2.安装Node.js

3.配置环境变量
CLAUDE_CODE_GIT_BASH_PATH = D:\Git\git-bash.exe

4.git bash中执行
# 使用npm安装Claude Code 改为国内可用的镜像地址
npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com
# 检查安装是否成功
claude --version

5.申请api key
以deepseek为例，新用户免费送1kw token
https://platform.deepseek.com/sign_in
得到api key如下
sk-a372692374c24f80990cb3973624149d
编辑文件 C:\Users\rbt\.claude\settings.json 内容如下
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://api.deepseek.com/anthropic",
    "ANTHROPIC_AUTH_TOKEN": "sk-a372692374c24f80990cb3973624149d",
    "ANTHROPIC_MODEL": "deepseek-v4-flash"
  }
}

6.hello world
进入工程，需要先执行git init
然后执行claude
